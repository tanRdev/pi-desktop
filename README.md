# PiDesk

PiDesk is a macOS-only Electron desktop wrapper for the Pi coding agent.

## Current scope

This workspace is being bootstrapped around three initial milestones:

1. monorepo + validation harness,
2. secure Electron shell,
3. hidden Pi agent host utility process.

## Development

```bash
pnpm install
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```
